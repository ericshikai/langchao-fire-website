Netlify CMS 后台集成指南

按照以下步骤，将网站部署到 Netlify 并启用可视化内容管理后台（完全免费）

⏱️ 预计完成时间：15–30 分钟  |  💰 费用：完全免费（Netlify 免费套餐） |  🛠️ 所需账号：GitHub + Netlify

📦 第一阶段：准备 GitHub 仓库

1注册 / 登录 GitHub

访问 github.com，注册或登录你的账号。

→ 前往 GitHub 注册
2创建新仓库

点击右上角 + → New repository，填写：

  • Repository name: langchao-fire-website
  • Visibility: Public（免费 Netlify CMS 需要公开仓库）
  • 不要勾选初始化 README

点击 Create repository。

3上传网站文件到仓库

将下载的网站文件夹内容上传到 GitHub 仓库：

方法一：网页上传（推荐，无需命令行）

在新建的仓库页面点击 uploading an existing file，将整个 website 文件夹内的所有文件拖入上传区域，然后点击 Commit changes。

方法二：Git 命令行

cd website
git init
git add .
git commit -m "Initial website commit"
git branch -M main
git remote add origin https://github.com/你的用户名/langchao-fire-website.git
git push -u origin main
⚠️ 确保 admin/ 文件夹、_data/ 文件夹和 netlify.toml 都包含在上传文件中。

🚀 第二阶段：部署到 Netlify

4注册 / 登录 Netlify

访问 netlify.com，使用 GitHub 账号登录（推荐）。

→ 前往 Netlify 注册
5从 GitHub 导入项目

在 Netlify 控制台：

  • 点击 Add new site → Import an existing project
  • 选择 Deploy with GitHub
  • 授权 Netlify 访问你的 GitHub 账号
  • 选择 langchao-fire-website 仓库

部署设置保持默认（无需构建命令），点击 Deploy site。

✅ 几分钟后网站即可在 Netlify 提供的域名下访问，如 langchao-fire.netlify.app
6（可选）绑定自定义域名

在 Netlify 站点设置 → Domain management → Add custom domain，输入你的域名并按提示修改 DNS 记录。

🔐 第三阶段：启用 CMS 登录认证

7开启 Netlify Identity 服务

在 Netlify 控制台，进入你的站点：

  • 点击顶部菜单 Integrations → 搜索 Identity
  • 点击 Enable Identity
  • 在 Identity 设置中，将 Registration 改为 Invite only（仅邀请，防止他人注册）
⚠️ 务必设置为 Invite only，否则任何人都可以注册后台账号！
8开启 Git Gateway

在 Netlify Identity 设置页面：

  • 滚动到 Services 部分
  • 点击 Enable Git Gateway

这允许 CMS 通过 Netlify 写入 GitHub 仓库，无需直接使用 GitHub 凭据。

9邀请自己为管理员

在 Netlify Identity 页面：

  • 点击 Invite users
  • 输入你的邮箱地址
  • 检查邮箱，点击邀请链接设置密码
✅ 完成后即可访问 https://你的站点.netlify.app/admin/ 登录后台！

🎨 第四阶段：更新 CMS 配置中的域名

10修改 admin/config.yml 中的网站地址

在 GitHub 仓库中编辑 admin/config.yml，将以下两行更新为你的实际 Netlify 域名：

site_url: https://你的站点名.netlify.app
display_url: https://你的站点名.netlify.app

保存后 Netlify 会自动重新部署（约1分钟）。

✅ 完成！如何使用后台

🖥️ 访问后台管理界面

后台地址：https://你的站点.netlify.app/admin/

  • 使用邀请时设置的邮箱和密码登录
  • 左侧导航可以管理：网站设置、首页内容、产品、认证、联系页面、新闻动态
  • 每次保存更改后，Netlify 会自动将修改推送到 GitHub 并重新部署网站
  • 部署通常在 1–2 分钟内完成
💡 可以邀请多个管理员账号，每位账号的操作都会以独立 Git commit 记录，方便追溯历史。
📋 后台功能说明
  • ⚙️ 网站设置 — 修改公司名称、联系方式、基本数据
  • 🏠 首页内容 — 修改 Banner 文字、Why Choose Us 内容和图片
  • 🧯 产品管理 — 新增/编辑/删除产品，上传产品图片
  • 🏆 认证管理 — 管理认证证书信息和图片
  • 📬 联系页面 — 修改联系信息和 FAQ 问答
  • 📰 新闻动态 — 发布行业新闻和公司动态

❓ 常见问题

登录时提示 "Git Gateway Error"？

请确认已在 Netlify Identity → Services 中点击了 Enable Git Gateway。

CMS 中修改后网站没有更新？

检查 Netlify 控制台的 Deploys 标签，查看是否有正在进行的部署或错误信息。

忘记后台密码？

在 Netlify Identity 页面找到对应用户，点击 Reset password 发送重置邮件。

能否使用免费套餐？

可以。Netlify 免费套餐包含：每月 100GB 流量、300 分钟构建时间、1 个并发构建，对于展示型企业网站完全足够。
